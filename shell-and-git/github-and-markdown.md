# github & markdown

## markdown basic commands

- `#` for heading
- `*` for italic
- `**` for bold
- `***` for bold and italic
- `---` for horizontal line
- `> ` for blockquote
- `* ` for unordered list
- `1. ` for ordered list
- `[]()` for link
- `![alt text](image link)` for image
- ` ` for code
- ` ``` ` for code block
- ` ```language ` for code block with language
